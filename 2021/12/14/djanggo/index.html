django

web项目：前端、后端

前端：html css js

后端（后台）：java/python mybatis,django:数据传递和存储（存储在数据库）

安装django:pip install django

创建项目：django-admin startproject 项目名

创建应用：先进入manage.py文件的同层文件夹中，命令：python manage.py startapp 应用名

运行项目：

# 1、打开manage.py文件
# 2、右键运行
# 3、配置参数：name：项目名称，parameter:runserver （端口号）

配置项目的应用

#1、在settings.py的INSTALLED_APPS中：注册应用,"demo"  (demo是应用名)
#2、总路由分发：urls.py :
            # 导入 from django.urls import include
            # path('demo/',include("demo.urls"))
#3、在应用中创建子路由urls.py，
            # 导入视图类：from . import views
#4、在应用的views.py中创建视图函数（网页）
            #导入 from django.http import HttpResponse
            # 定义视图函数
                #def index(request):
                    #return HttpResponse("视图内容")
#5、在子路由urls.py文件中添加视图路由
            # path('index',views.index) index:视图路由，views.index:视图函数
